Card Memory

Es un juego web interactivo de memoria inspirado en personajes del universo Marvel. El juego desafía a los usuarios a encontrar pares de cartas iguales en el menor tiempo posible. Cuenta con una interfaz dinámica, efectos visuales llamativos y una experiencia responsive, ideal para usuarios de todas las edades que buscan entretenimiento casual.

Rick y Morty

Es una web que permite explorar personajes de la famosa serie Rick and Morty. Desarrollada con Angular y conectada a la API pública de Rick and Morty, ofrece funcionalidades como búsqueda en tiempo real, visualización de detalles por personaje, e indicadores visuales de estado, género y ubicación. Presenta un diseño responsive y atractivo.

Country Web

Es una aplicación web interactiva desarrollada con Angular que permite buscar y explorar información detallada de países de todo el mundo. Utiliza la API de REST Countries para obtener datos en tiempo real, como la bandera, capital, región, etc. Incluye funciones de búsqueda dinámica, visualización responsive y navegación sencilla a través de rutas.
